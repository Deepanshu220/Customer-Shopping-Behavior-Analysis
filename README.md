# Customer-Shopping-Behavior-Analysis
End-to-end analysis of customer shopping behavior using Python, SQL, and Power BI.

## Objectives
- Analyze customer purchasing behavior.
- Compare subscriber and non-subscriber spending.
- Study the impact of discounts.
- Segment customers based on purchase history.
- Build an interactive Power BI dashboard.

## Tools Used
- Python
- SQL
- Power BI

## Dataset
- 3,900 purchase records
- 18 variables
- 37 missing values in the Review Rating column

## Key Business Questions
- Revenue by Gender
- High-Spending Discount Users
- Top Rated Products
- Shipping Type Comparison
- Subscribers vs Non-Subscribers
- Discount-Dependent Products
- Customer Segmentation
- Top Products by Category
- Repeat Buyers and Subscription Behavior
- Revenue by Age Group
